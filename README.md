# Text to Speech Converter

A simple and elegant **Text-to-Speech Converter** built using **HTML, CSS, and JavaScript**.  
This web app allows users to type any text, choose a voice, and listen to the spoken output using the Web Speech API.

---

## Features

-  User-friendly text input area  
-  Converts written text into spoken audio  
-  Multiple voice options (depends on browser support)  
-  Modern, responsive UI  
-  Fast and lightweight  
-  Works directly in the browser (no backend needed)

---

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Web Speech API)**

---

## How to Use

1. Type any text into the text box  
2. Select your preferred voice from the dropdown menu  
3. Click the **Listen** button  
4. The browser will speak the text aloud
